# АрлекинО — социальная платформа мастеров

Клон соцсети в стиле ВКонтакте/Одноклассники с цирково-готическим дизайном.  
PHP 8+, MySQL.

## Роли

- **Админ-мастер** — управление сайтом из админки: пользователи, настройки, бан, верификация мастеров.
- **Мастер** — личный кабинет: публикация работ (фото, видео, текст), просмотр рейтинга и личных сообщений.
- **Клиент** — просмотр страниц мастеров, их работ, рейтинга и отправка личных сообщений мастерам.

## Установка (MySQL)

1. Скопируйте `.env.example` в `.env`:
   ```bash
   cp .env.example .env
   ```

2. Создайте пользователя и БД в MySQL:
   ```bash
   sudo mysql < install/create-db-user.sql
   ```
   Создаётся БД `circus_social` и пользователь **circus** / пароль **admin123**.

3. Установите зависимости (если есть composer):
   ```bash
   composer install
   ```

4. Создайте таблицы и админа:
   ```bash
   php install/install.php
   php install/migrate_v2.php
   ```

5. Запуск:
   ```bash
   php -S 0.0.0.0:8000 -t public
   ```
   Сайт: http://localhost:8000

**Админ по умолчанию:** admin@circus.local / admin123

Подробнее: см. [DEPLOY.md](DEPLOY.md)

## Альтернатива: setup-mysql.sh

Одной командой: `./install/setup-mysql.sh` — проверка MariaDB, установка php-mysql, создание таблиц, миграция, запуск сервера.

## Устранение проблем

- **Unit mysql.service not found** — в Ubuntu часто стоит MariaDB. Запустите: `sudo systemctl start mariadb`. Если службы нет: `sudo apt-get install -y mariadb-server`
- **403 Forbidden при apt update** (репозиторий Cursor) — временно отключите его:  
  `sudo mv /etc/apt/sources.list.d/cursor*.list /etc/apt/sources.list.d/cursor.list.bak 2>/dev/null; sudo apt-get update`  
  Затем установите вручную: `sudo apt-get install -y php-mysql` (или `php8.3-mysql`), потом снова включите репозиторий при необходимости.
- После установки драйвера и запуска MariaDB/MySQL выполните вручную:  
  `php install/install.php`  
  затем запуск сервера: `php -S 0.0.0.0:8000 -t public`

## Структура

- `config/` — конфигурация
- `database/schema.sql` — схема БД
- `public/` — точка входа, CSS, загрузки
- `src/` — классы приложения (Auth, Database, репозитории)
- `templates/` — шаблоны страниц
- `install/install.php` — установка БД и админа

## Работа сайта

- **Главная** — лента записей всех мастеров и блок «Мастера».
- **Мастера** — список мастеров с рейтингом.
- **Страница мастера** — профиль, отзывы/рейтинг (клиенты могут ставить оценку 1–5), работы (посты с медиа). Кнопка «Написать сообщение» для клиентов.
- **Личный кабинет** (только мастер/админ) — редактирование профиля, новая запись (текст + фото/видео), удаление записей, просмотр отзывов.
- **Сообщения** — личные диалоги клиент ↔ мастер.
- **Админка** (только админ) — настройки сайта (название, описание, записей на странице, разрешить регистрацию), список пользователей: бан/разбан, верификация мастеров.

Дизайн: тёмная тема, шрифты Cinzel и Cormorant Garamond, акценты красный и золотой (цирк + готика).
